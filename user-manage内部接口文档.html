接口1

请求地址 : ‘${ctx}/login’
请求方式 : POST
请求参数及示例
请求参数 参数类型 参数说明
userName String,不可为空 当前登录用户的用户名
userPassword String,不可为空 当前登录用户的密码 
{
  "userName": "lucy",
  "userPassword": "123456",
}
返回参数及示例
返回参数 参数类型 参数说明
status String,不可为空 请求执行状态，0表示执行成功，1表示用户名或者密码格式有误，2表示用户名及密码验证失败，3表示服务器执行失败
userId String,不可为空 当前用户id
userName String,不可为空 当前用户名
userRealName String,不可为空 当前用户备注名
userInfos json对象 详细示例如下

接口4

请求地址 : ‘${ctx}/ modify-user’
请求方式 : POST
请求参数及示例
请求参数 参数类型 参数说明
userId String,不可为空 修改用户的用户id
userName String 用户名，如果为空表示不修改
userRealName String 用户备注名，如果为空表示不修改
userPassword String 当前登录用户的密码，如果为空表示不修改 
{
  "userId": "7",
  "userName": "",
  "userRealName": "李达",
  "userPassword": "",
}
返回参数及示例
返回参数 参数类型 参数说明
status String,不可为空 请求执行状态，0表示执行成功，1表示用户名或者密码格式有误，2表示用户名及密码验证失败，3表示服务器执行失败 
{
  "status": "0",
}
